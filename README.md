# Trabalho - Indexação (Hash Extensível e Árvore B+)

Repositório do trabalho da disciplina **SGBD**.

## Objetivo

Implementar e comparar duas estruturas de indexação:

- **Hash Extensível**

- **Árvore B+**

- **Árvore ISA**

Com as operações: inserir, buscar, remover e exibir.

## Linguagem

O projeto desenvolvido em **Python 3.x**.

## Estrutura do Projeto

```
csgbd/
├── BPlusTree/
│   └── BPlusTree.py
├── ExtensibleHash/
│   └── ExtensibleHash.py
└── ISA/
    └── ISATree.py
```

## Execução

### Hash Extensível

```bash
python ExtensibleHash/ExtensibleHash.py
```

### Árvore B+

```bash
python BPlusTree/BPlusTree.py
```

### Árvore ISA

```bash
python ISA/ISATree.py
```
